### 학습 방식

- 각 단원별 한명씩 지정, 해당 인원이 이해하고 학습한 내용을 바탕으로 다른 팀원들에게 개념 설명 및 정리
- 다른 팀원들은 해당 개념에 대하여 모르는 내용 질의응답
- 5 Chapter가 끝날때 마다, 각 단원별 지정된 인원이 발표를 실시함
- 끝나는 시점이 이른 경우, 해당 단원이 끝난 날 진행하고, 늦은 경우 다음 날 오전에 실시
